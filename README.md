## 介绍

本项目是一款用于自动检测项目【控制器内存泄漏】的工具，如果你想所以，只需要把 `https://github.com/liangdahong/BMeaksFinder/tree/master/BMeaksFinder/Sources` 路径下的文件拖入项目即可，效果如下，在上线时请务必移除框架，目前处于开发尝试阶段，所以功能比较单一，如果你感兴趣可以一起聊聊。
